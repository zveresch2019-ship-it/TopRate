Privacy Policy

TopRate

Last updated: November 5, 2025

1. Introduction

This Privacy Policy describes how the "TopRate" application (hereinafter - "Application") processes user information.

Important: The TopRate application does not collect, store on external servers, or transmit any personal user data to third parties.

2. Data Collection

The Application collects minimal necessary information for operation:

  • Username and password - for authentication in the application
  • Player data and ratings - stored on the server for synchronization between devices
  • Match results - saved on the server

The Application DOES NOT collect the following information:

  • Personal data (name, surname, address, phone)
  • Email addresses (except username for login)
  • Geolocation
  • Device data
  • IP addresses (logged only for technical purposes)
  • Cookies or tracking identifiers
  • Usage analytics

3. Data Storage

Data is stored on a secure server and includes:

  • Username and hashed password
  • List of players and their ratings (linked to your group)
  • Match results (linked to your group)
  • Game statistics
  • Group and season information

Data security:

  • Passwords are stored in encrypted form (bcrypt)
  • Data is accessible only to members of your group
  • Data is not shared with third parties
  • You can delete your data at any time

4. Data Usage

Collected data is used exclusively for:

  • User authentication and authorization
  • Synchronizing data between devices
  • Displaying the list of players and their ratings
  • Calculating ratings based on match results
  • Saving match history
  • Displaying statistics

5. Third-Party Data Sharing

The Application DOES NOT share data with third parties, including:

  • Advertising networks
  • Analytics services
  • Social networks
  • Any other external services

Data is stored on a secure server (Railway) and is accessible only to members of your group.

6. Internet Connection

The Application requires an internet connection to function, as data is stored on the server for synchronization between devices.

7. Advertising

The Application does not contain advertising and does not use advertising identifiers.

8. Children's Safety

The Application is suitable for users of all ages (4+). We do not collect personal data from children or adults.

9. Your Rights

You have full control over your data:

  • You can delete any data within the application
  • You can export data for backup
  • You can delete all data by uninstalling the application

10. Policy Changes

We may update this Privacy Policy. In case of significant changes, we will notify you through an application update.
